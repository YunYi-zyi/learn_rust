# Learn Rust

这是我的 Rust 语言学习项目，用于记录和实践 Rust 编程语言的各种概念和功能。

## 项目介绍

本项目是一个学习 Rust 编程语言的实践仓库，包含了各种 Rust 语言特性的示例代码和练习。

## 项目结构

```
learn_rust/
├── src/
│   └── main.rs          # 主程序入口
├── Cargo.toml           # 项目配置文件
├── Cargo.lock           # 依赖锁定文件
└── README.md            # 项目说明文档
```

## 运行项目

确保您已经安装了 Rust 和 Cargo，然后在项目根目录下运行：

```bash
# 编译并运行项目
cargo run

# 仅编译项目
cargo build

# 运行测试
cargo test
```

这个项目将涵盖以下 Rust 语言特性：

- [ ] 错误处理
- [ ] 泛型和 Trait
- [ ] 生命周期
